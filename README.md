# Calculator

A simple calculator web app built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Clear and delete actions
- Responsive and clean interface

## Project Files
- index.html: Main structure of the calculator
- style.css: Styling for the calculator layout and buttons

## How to Run
1. Open index.html in your browser.
2. Or use a simple local server if you prefer.

## Notes
This project is meant as a beginner-friendly example of a web-based calculator.
